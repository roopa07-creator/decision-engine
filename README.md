# 🚀 Decision Engine (Flask Project)

A Flask-based Decision Engine that provides recommendations based on user inputs like salary, risk level, and location.

## 🔧 Features
- REST API using Flask
- User-friendly UI
- Decision logic engine
- Postman testing support
